# Installation

If you want to test the app by running it locally in PC, follow these instructions:
1.Clone the repo:

```
git clone https://github.com/UttamMarandi/Mr.Portor-E-Commerce-Clone.git
```

2.Install the packages :

```
yarn install
```

3.Start local server

```
yarn start
```
